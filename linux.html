Настройка для Linux


  1.     Рассмотрим на примере операционной
        системы Ubuntu 20.04.
  2.     На текущий момент Ubuntu не имеет
        инструментов в графическом интерфейсе
        для работы с протоколом WireGuard,
        поэтому настройку будет необходимо
        выполнять через терминал.
  3.     1. Установите WireGuard: 
  4. apt install wireguard
  5.     2. Откройте файл wg0.conf и скопируйте в
        него содержимое конфигурационного
        файла, который пришел на вашу почту после     установки сервера VPN. 
  6. nano /etc/wireguard/wg0.conf
  7.     3. Сохраните изменения.
  8.     Для подключения используйте команду: 
  9. wg-quick up wg0
  10.     Если нужно, чтобы подключение по
        WireGuard устанавливалось автоматически
        при запуске системы, добавьте wg-quick в
        автозагрузку: 
  11. systemctl enable wg-quick@wg0.service
