Инструкция




Настройка для MacOS


  1.     1. Загрузите приложение Wireguard из
        официального сайта     или AppStore.
  2.     2. Скачайте конфигурационный файл,
        полученный в телеграм-боте.
  3.     3. Откройте приложение WireGuard и
        выберите «Управлять туннелями Wireguard».
  4.     4. Нажмите кнопку «Импорт туннелей из
        файла».

    5.     5. Выберите скачанный ранее
        конфигурационный файл .conf. Далее
        отобразится окно подтверждения на
        разрешение конфигурации.

  5.     6. Выберите «Разрешить»
        (конфигурационный файл будет добавлен в
        исключения брандмауэра).
  6.     7. Готово. Для подключения к VPN нажмите
        кнопку «Подключить».
